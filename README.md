## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/elifnurkarakoc/responsive-viewer-app.git
   ```
2. Install packages
   ```sh
   yarn
   ```
3. Run the development server:

   ```bash
   yarn dev
   ```
